🛒 GreenCart - MERN Grocery Delivery Platform



📌 Overview

GreenCart is a full-stack MERN grocery delivery application that allows users to browse products, manage their cart, place orders, and make secure online payments. The platform also includes a seller dashboard for managing products and orders.

It is built using the MERN Stack with secure authentication, Cloudinary image storage, Stripe payment integration, and a responsive user interface.
